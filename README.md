# mac

Update all packages:
```bash
brew update
brew upgrade
```

Install regular tools:
```bash
brew install \
  go \
  multipass \
  ansible \
  hugo \
  kubectx \
  awscli \
  node \
  helm \
  iproute2mac \
  wget \
  bat \
  azure-cli
```

Delete windscribe data:
```bash
rm ~/Library/Preferences/com.windscribe.Windscribe.plist
```
